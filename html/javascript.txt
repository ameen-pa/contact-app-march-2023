                                           Javascript - In Front end (Scripting language)
                                  -------------------------------------------------------------     
                                        1. Document Object Model (DOM) - document
                                        2. Methods to apply to html
                                           - Internal JS - use script tag
                                           - External JS - use js file
                                        3. DOM Methods: Access HTML elements in JS
                                           - Using tag name: document.getElementByTagName('tag')
                                           - Using ID: document.getElementById('id')
                                           - Using class name: document.getElementByClassName('class-name')
                                           - Using QuerySelector: document.querySelector('tag/#id/.class-name')
                                           - Using QuerySelectorAll: document.querySelectorAll('tag/#id/.class-name')
                                        4. To Access / Update content in an html elements
                                           - innerHTML
                                           - textContent
                                        5. Event - Actions performed by user to trigger the update in DOM
                                           - click
                                           - KeyDown
                                           - KeyUp
                                           - submit
                                           - change
                                           - load
                                           - resize
                                           - scroll
