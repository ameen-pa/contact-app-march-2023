DOM - Document Object Model

DOM Methods

  1. Using tag name: document.getElementByTagName('tag')
  2. Using ID: document.getElementById('id')
  3. Using class name: document.getElementByClassName('class-name')
  4. Using QuerySelector: document.querySelector('tag/#id/.class-name')
  5. Using QuerySelectorAll: document.querySelectorAll('tag/#id/.class-name')
